## For private use only. Unauthorized use or distribution is strictly prohibited.

初始化仓库
```bash
git init
````

添加所有文件
```bash
git add .
```

修改分支名称
```bash
git branch -m ""
```

使用 rebase 策略合并代码
```bash
git pull --rebase origin main
```

提交 commit 信息
```bash
git commit -m ""
```

推送至远程仓库
```bash
git push origin main
```
